🌟 このレッスンについて

「アプリ開発」と聞くと、多くの方がこう思うかもしれません。「自分には無理だ。プログラミングって難しいんでしょ？」

でも、ちょっと待ってください。今はもう、その常識が変わっています。

コードを書けなくても、アプリが作れる時代

2024年以降、AIの急速な進化により、プログラミングの世界に革命が起きました。以前はエンジニアが何ヶ月もかけて作っていたようなアプリが、AIと会話するだけで数時間〜数日で完成するようになったのです。

このプログラムで使う「Claude Code」は、まさにその革命の中心にあるツールです。あなたが日本語で「こんなアプリを作りたい」と伝えるだけで、Claude Codeが実際のコードを書き、ファイルを作り、問題を解決してくれます。あなたはコードを一行も書く必要がありません。

📋 手順

手順1｜Git for Windowsをインストールする

Gitとは何か？
Gitは「ファイルの変更履歴を管理するツール」です。Gitをインストールする主な目的は、「Git Bash」を手に入れることです。

Git Bashとは何か？
Git Bashは、パソコンに文字で指示を出すための「黒い画面」です。Claude Codeのインストールや起動は、この画面を使って行います。やることは決まった文字をコピー＆ペーストするだけです。

💡 「ターミナル」や「コマンドプロンプト」と呼ばれるものと同じ種類のツールです。Git Bashは日本語環境でも扱いやすく、このスクールではGit Bashを使って作業を進めます。
  1. https://git-scm.com にアクセスする
  2. 「Download for Windows」をクリックする
  3. ダウンロードされた .exe ファイルを開く
  4. インストール画面はすべて「Next」で進んでOK
  5. 最後に「Install」→「Finish」をクリック
✅ 確認方法：スタートメニューで「Git Bash」と検索して、アプリが表示されればインストール成功です。

手順2｜Claude Codeをインストールする

① Git Bashを開く

  1. 画面左下のスタートボタン（Windowsマーク）をクリックする
  2. キーボードで「Git Bash」と入力して検索する
  3. 「Git Bash」というアプリが表示されたらクリックして開く
💡 黒い画面（ターミナル）が開けば成功です。マウスでは操作できませんが、決まった文字を入力するだけなので慣れれば難しくありません。

② コマンドを貼り付けてインストールする

以下のコマンドをコピーして、Git Bashの黒い画面を右クリック → 「Paste」で貼り付けます。

⚠️ Git Bashでは通常の Ctrl+V では貼り付けできません。右クリック → 「Paste」を使ってください。
npm install -g @anthropic-ai/claude-code
💡 文字がたくさん流れ始めるので、止まるまで待ちましょう（1〜3分程度）。インストール中は何も操作しなくて大丈夫です。

③ インストールできたか確認する

claude --version

「1.x.x」のような数字が表示されればインストール成功です。

⚠️ 「npm: command not found」と表示された場合：Node.jsがインストールされていません。https://nodejs.org にアクセスし、「LTS」と書かれた緑色のボタンからダウンロードしてインストールしてください。インストール後はGit Bashを再起動してから再試行してください。

手順3｜Claude Codeを起動して日本語で話しかける

  1. Git Bashで、作業用フォルダに移動する
cd ~/Documents mkdir my-app cd my-app
💡 コマンドの意味：
• cd ~/Documents：「ドキュメント」フォルダに移動します。cdは「Change Directory（フォルダを移動する）」の略です。~（チルダ）は「自分のホームフォルダ」を表す記号で、Windowsでは C:¥Users¥あなたのユーザー名 に相当します。別の場所で作業したい場合は、~/Documentsの部分を書き換えてください（例：デスクトップなら cd ~/Desktop）。
• mkdir my-app：「my-app」というフォルダを新しく作ります。
• cd my-app：作ったフォルダの中に移動します。
  1. Claude Codeを起動する
claude
  1. 初回は認証画面が表示されます。ブラウザが開くので、Claude.aiアカウントでログインしてください。
  2. 起動できたら、日本語で話しかけてみましょう
こんにちは。私はプログラミング初心者です。よろしくお願いします。
💡 Claude Codeを終了するときは Ctrl + C を押すか、/exit と入力してください。

手順4｜CLAUDE.mdファイルを作成する

CLAUDE.mdは、Claude Codeに「自分のことを覚えておいてほしい情報」を書いておくファイルです。Claude Codeに以下のように頼んでみましょう。

CLAUDE.mdファイルを作成してください。 内容は以下を参考にしてください： ## 私について - 名前：（あなたの名前） - 目標：初心者向けのシンプルなWebアプリを作ること - 使用言語：HTML、JavaScript（これから学ぶ） ## お願い - コードには必ずコメントを書いてください - 説明は初心者にもわかりやすい言葉でお願いします - エラーが出たときは原因と解決方法も教えてください
🚨 よくあるトラブル
「npm: command not found」と表示される
Node.jsがインストールされていません。https://nodejs.org にアクセスし、「LTS版」をダウンロードしてインストールしてください。インストール後はGit Bashを再起動してから再試行してください。
Claude Codeのインストール後に「claude: command not found」になる
Git Bashを一度閉じて再起動してください。それでも解決しない場合は以下を試してください：
export PATH="$HOME/.npm-global/bin:$PATH"
認証画面（ブラウザ）が開かない
Claude Codeが表示するURLをコピーして、手動でブラウザに貼り付けてください。
日本語が文字化けする
Git Bashのウィンドウを右クリック →「Options」→「Text」→「Locale」を ja_JP、「Character set」を UTF-8 に変更してください。

🌟 このレッスンについて

レッスン1では、Claude Codeを起動して日本語で話しかけることができました。それだけで、すでに大きな一歩です。

レッスン2のゴールは、自分のプロフィールページを作って、インターネット上に公開することです。

HTMLって何？

ウェブページはすべて、「HTML（エイチティーエムエル）」というファイルで作られています。このレッスンではHTMLを自分で書く必要はありません。Claude Codeに「こんなページを作って」と日本語で頼むだけで大丈夫です。

Vercelって何？

Vercel（バーセル）は、作ったウェブページを無料でインターネットに公開できるサービスです。GitHubと連携させることで、ファイルをアップロードするだけで自動的に公開URLが発行されます。

必要なもの

  • レッスン1が完了していること（Claude Codeが起動できる状態）
  • GitHubのアカウント（まだの方は https://github.com で無料作成）
  • Vercelのアカウント（まだの方は https://vercel.com でGitHubアカウントを使ってサインアップ）
  • 作業時間の目安：初回は1〜2時間
📋 手順

手順1｜新しいフォルダを作る

  1. Git Bashを開く
  2. 以下のコマンドを実行する
cd ~/Documents mkdir profile-page cd profile-page
💡 コマンドの意味：
• cd ~/Documents：「ドキュメント」フォルダに移動します。~（チルダ）は「自分のホームフォルダ」を表す記号で、Windowsでは C:¥Users¥あなたのユーザー名 に相当します。別の場所で作業したい場合は、~/Documentsの部分を書き換えてください（例：デスクトップなら cd ~/Desktop）。
• mkdir profile-page：「profile-page」というフォルダを新しく作ります。
• cd profile-page：作ったフォルダの中に移動します。
  1. Claude Codeを起動する
claude

手順2｜Claude Codeにプロフィールページを作ってもらう

名前：（あなたの名前）、職業：（あなたの職業） という情報で、シンプルなプロフィールページを作ってください。 背景は白で、読みやすいデザインにしてください。 HTML��CSSを使い、1つのファイルにまとめてください。

しばらく待つと、index.html というファイルが作成されます。エクスプローラーで Documents > profile-page フォルダを開き、index.html をダブルクリックするとブラウザでページが表示されます。

💡 気に入らないところがあればClaude Codeにそのまま伝えましょう。「名前の文字をもっと大きくしてください」「背景色を薄いベージュにしてください」など、何度でも修正してくれます。

手順3｜GitHubにアップロードする

  1. https://github.com にログインする
  2. 右上の「＋」ボタン →「New repository」をクリック
  3. 「Repository name」の入力欄に profile-page と入力する
  4. 「Public」が選択されていることを確認して、「Create repository」ボタンをクリック
  5. 「Quick setup」という画面が表示される
  6. 画面の中ほどに小さく表示されている文章を探す
    → 「uploading an existing file」 という青いリンクをクリックする
⚠️ ここが迷いやすいポイント：目立つ大きなボタンではなく、文章の中の小さなリンクです。
  1. 点線で囲まれたエリアに index.html をドラッグ＆ドロップする
  2. 画面を下にスクロールして「Commit changes」という緑色のボタンをクリックする

手順4｜Vercelで公開する

Vercelのアカウントをまだ持っていない方

  1. https://vercel.com にアクセスする
  2. 画面右上の「Sign Up」をクリックする
  3. 「Continue with GitHub」をクリックする
  4. GitHubのログイン画面が表示されたらIDとパスワードを入力する
  5. 「Authorize Vercel」ボタンをクリックする
  6. プラン選択画面で「Hobby（無料）」を選んで進む
💡 Vercelの無料プランで今回の作業はすべてできます。クレジットカードの入力は不要です。

プロジェクトを作成して公開する

  1. 「Add New...」ボタン → 「Project」を選ぶ
  2. 「Import Git Repository」でさきほど作った profile-page を探して「Import」をクリック
  3. 設定はそのままで「Deploy」ボタンをクリックする
  4. 「Congratulations!」画面が表示されたら公開完了！
✅ 発行されたURL（例：https://profile-page-xxx.vercel.app）をスマホで開いても同じページが表示されます！

🌟 このレッスンについて

レッスン1〜3で、環境構築・HTML・JavaScriptの基本を体験しました。レッスン4は集大成のレッスンです。それらのスキルを全部使って、自分のアイデアでゲームアプリを一から作り、世界に公開します。

なぜゲームなのか

  • 後で検証できる：データベース不要で、ゲームができるのかできないのか明確
  • 楽しい：自分が完成したもので遊びながらテストできる
  • 伝わる：完成したURLを家族や友人に送るだけで反応が返ってくる
💡 完璧を目指さなくてOK。「動く」ことが最優先。バグはエラーメッセージをそのままClaude Codeに貼り付ければ直してくれます。

ゲームアイデア例

  • 漢字クイズゲーム（選択式で正解を選ぶ）
  • 計算ドリル（足し算・引き算・掛け算・割り算をランダム出題）
  • 都道府県クイズ（県庁所在地を当てる）
  • 記憶力トレーニング（カードを使って覚える）
  • タイピング練習ゲーム

ウェブページやアプリをインターネットに無料で公開できるサービス。GitHubと連携するだけで自動的にURLが発行される。
デプロイ
作ったアプリをインターネット上に公開する操作のこと。「公開する」と同じ意味で使われることが多い。
リポジトリ
GitHubでファイルを管理するための入れ物。プロジェクトごとに1つ作るのが基本。
バグ
プログラムの不具合のこと。意図したとおりに動かない現象を指す。
レスポンシブデザイン
スマホ・タブレット・パソコンなど、画面サイズに合わせて自動的にレイアウトが調整されるデザインのこと。
CLAUDE.md
Claude Codeに「自分のことを覚えておいてほしい情報」を書いておくファイル。毎回説明しなくてもよくなる。
